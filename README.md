# EMI Calculator

A responsive EMI Calculator built using React and Vite.

## Features

* Calculate Monthly EMI
* Calculate Total Interest Payable
* Calculate Total Repayment Amount
* Responsive UI
* Instant calculations

## Tech Stack

* React
* JavaScript
* CSS
* Vite

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
